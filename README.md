# cli_notes-app
This is a notes app created using nodejs which uses command line to parform operation add, remove, list, read.
- To Add <br>
- node app.js add --title="t" --body="" <br>
- To Remove <br>
- node app.js remove --title="t" <br>
- To List <br>
- node app.js list <br>
- To Read <br>
- node app.js read --title="t" <br>
